# 时间感校准实验室 — 设计文档 v1

## 项目概念

「你的内在时钟，偏了多少？」

通过三轮不同时长的时间感知测试，量化用户的主观时间与客观时间的偏差，生成时间人格报告。

---

## 核心玩法

- 用户在布朗运动粒子背景下，凭直觉感受指定时长，觉得"到了"就点击屏幕
- 三轮测试，每轮时长不同（5-20秒区间，相邻差值≥4秒，顺序打乱）
- 每轮背景主题不同，避免同一视觉对计数产生影响
- 背景采用布朗运动微粒：无节奏、无周期、无法被用作计时参照

## 交互细节

1. **禁止下拉刷新和系统手势**：CSS `overscroll-behavior: none` + `touch-action: manipulation`
2. **屏蔽双击缩放**：`touch-action: manipulation`
3. **提示音用 Web Audio API 预加载**：避免首次播放延迟
4. **计时精度用 `performance.now()`**：不用 `Date.now()`
5. **时长生成逻辑**：5-20秒区间随机取三个值，相邻差值≥4秒，不按升序排列

## 三轮布朗运动视觉主题细化

### 第 1 轮：深海浮游

- **背景色**：#0a1628 → #0d2b3e 径向渐变（中心稍亮）
- **粒子**：30-50 个，半径 2-8px 不等，颜色 hsl(170-200, 60-80%, 50-70%)
- **运动**：速度极慢（0.1-0.3 px/frame），方向每帧加微量随机偏移
- **细节**：粒子 20-60% 不透明度，大粒子更透明（模拟景深）；缓慢明灭

### 第 2 轮：宇宙尘埃

- **背景色**：纯 #000000
- **粒子**：80-120 个，半径 1-3px，颜色 hsl(30-50, 40-80%, 70-90%)（暖白到金色）
- **运动**：速度 0.2-0.5 px/frame，方向完全随机
- **细节**：5% 粒子偶尔闪烁；极微弱 noise grain 叠加

### 第 3 轮：微观分子

- **背景色**：#0f0f1a（极深紫黑）
- **粒子**：40-60 个，半径 3-5px，颜色 hsl(280-340, 70%, 60-80%) + 少量青绿点缀
- **运动**：速度 0.5-1.0 px/frame，边缘反弹，粒子间简单斥力
- **细节**：距离 < 80px 时画极淡连线（opacity 0.05-0.1）

## 历史人物匹配系统（九宫格）

### 判定逻辑

```
平均偏差率 = mean(各轮偏差百分比)  // 正值=用户点早了=感觉时间快
波动度 =max(各轮偏差率) - min(各轮偏差率)

快慢维度：
  > +10%  → 偏快
  -10% ~ +10% → 居中
  < -10% → 偏慢

稳定维度：
  < 12%  → 精密稳定
  12% ~ 28% → 中等波动
  > 28%  → 混沌波动
```

### 九位人物

| 快慢 \ 稳定性 | 精密稳定 | 中等波动 | 混沌波动 |
|---|---|---|---|
| **偏快** | 拿破仑 | 尼古拉·特斯拉 | 毕加索 |
| **居中** | 康德 | 诸葛亮 | 莫扎特 |
| **偏慢** | 达尔文 | 村上春树 | 李白 |

### 人物卡片文案

- **拿破仑**「精密加速者」— "他用 4 小时睡眠换取帝国版图，每一秒都被精确压榨。你的内在时钟是一台校准过的快表——始终比世界快半拍，但从不出错。"

- **毕加索**「时间的立体派」— "他把一天活成五天，但没人知道下一笔落在哪里。你的时间感像他的画——打碎了重组，每一刻都是即兴。"
- **康德**「人形原子钟」— "柯尼斯堡的居民用他的散步时间对表。你的时间感精确得令人不安——你确定自己不是一台仪器？"
- **诸葛亮**「借东风的人」— "他算天时、度地利、量人心，对时机的拿捏从不急躁也不迟钝。你的时间感如同一位善弈者——大局了然，偶有妙手。"
- **莫扎特**「即兴变奏曲」— "他 5 岁作曲，35 岁燃尽，时间准不准对他毫无意义——因为每一刻都在创造。你的时间不快不慢，但从不重复。"
- **达尔文**「地质纪年者」— "他花 20 年写一本书，像地层一样缓慢而确定地堆积证据。你的内在时钟比现实慢半拍，但这份从容是一种罕见的定力。"
- **村上春树**「长跑节拍器」— "每天凌晨 4 点起床，用重复抵抗混沌。你的时间流得比别人慢一些，像长距离跑者的呼吸——稳中带着微微的起伏。"
- **李白**「醉中仙」— "他举杯邀月、对影成三人，时间在他那里是一壶酒的事。你的内在时钟散漫、自在、不可捉摸——天地间一蜉蝣，何必计较快慢。"

### 粒子聚合动效

- 结果揭晓时粒子从布朗运动状态聚合成对应人物的符号化剪影
- 每个人物用 1-3 条极简 SVG path 表达标志性特征，统一画布 300×400
- 从 path 上等距采样 250 个点作为粒子目标坐标
- 动画节奏：0.3s 收缩到中心 → 0.2s 爆散 → 1s 聚合成图形
- 聚合完成后粒子在目标点 ±1-2px 微幅抖动

## 设计原则

- 计时态画面不能有任何周期性、节奏性视觉元素
- 布朗运动微粒：无方向、无周期、无节奏，大脑无法提取时间信息
- 起始标记用提示音而非倒计时，避免给用户注入时间刻度
- 三轮时长不同，防止用��用上一轮体感校准下一轮
