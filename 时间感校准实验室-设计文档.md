# 时间感校准实验室 — 设计文档 v2

## 项目概念

「你的内在时钟，偏了多少？」

通过三轮不同时长的时间感知测试，量化用户的主观时间与客观时间的偏差，生成时间人格报告并匹配历史人物。

## 核心玩法

- 三轮测试，每轮时长不同（5-20秒区间，相邻差值≥4秒，顺序打乱）
- 用户在动效背景下凭直觉感受时间，觉得"到了"就点击屏幕
- 每轮动效主题不同，但视觉风格统一（浅色系线条/几何/点阵）
- 关：动效无节奏、无周期，不能被用作计时参照

## 三轮动效主题（v2 实现版）

| 轮次 | 名称 | 形态 | 特征 |
|------|------|------|------|
| 1 | 涟漪 Ripple | 缓慢漂浮的细线段 | 20条线段，无规则方向漂移，opacity 0.08-0.2 |
| 2 | 脉冲 Pulse | 同心圆扩散消失 | 5个圆从随机位置生成并扩散淡出 |
| 3 | 漂移 Drift | 点阵网格随机微偏移 | 12×16 网格点在原位做随机位移 |

所有动效统一用 rgba(26,26,26, 低opacity) 在浅色背景上绘制。

## 历史人物匹配系统（九宫格）

### 判定逻辑

```
平均偏差率 = mean(各轮 (elapsed - target) / target × 100)
波动度 = max(各轮偏差率) - min(各轮偏差率)

快维度（用户点早了 = elapsed < target = 偏差率为负 = 感觉时间快）：
  偏差率 < -10%  → 偏快
  -10% ~ +10%   → 居中
  > +10%        → 偏慢

稳定维度：
  波动度 < 12%  精密稳定
  12% ~ 28%    → 中等波动
  > 28%        → 混沌波动
```

### 九位人物

| 快慢 \ 稳定性 | 精密稳定 | 中等波动 | 混沌波动 |
|---|---|---|---|
| **偏** | 拿破仑 | 特斯拉 | 毕加索 |
| **居中** | 康德 | 诸葛亮 | 莫扎特 |
| **偏慢** | 达尔文 | 村上春树 | 李白 |

### 粒子聚合文字动效

- 结果揭晓时，canvas 先渲染人物名字获取像素坐标
- 300个粒子从随机位置出发，通过 ease-out cubic 缓动归位到文字轮廓
- 90帧完成聚合，之后在目标点 ±0.8px 微幅抖动

## 交互细节

1. CSS `overscroll-behavior: none` + `touch-action: manipulation` 禁止误触
2. Web Audio API 预加载提示音（开始880Hz，结束660Hz）
3. `performance.now()` 高精度计时
4. 时长生成：5-20秒随机取三值，相邻差≥4秒，不按升序排列
5. 页面过渡动画：opacity + translateY，0.8s cubic-bezier(0.4, 0, 0.2, 1)

## 技术栈

- 纯 HTML + CSS + Vanilla JS（单文件 H5，无框架依赖）
- Google Fonts: Noto Serif SC + Playfair Display
- Canvas 2D API 绘制动效和粒子聚合
- 移动端优先设计，PC 端通过 max-width 约束
