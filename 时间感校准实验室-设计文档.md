# 时间感校准实验室 — 设计文档 v3.2

## 项目概念

「你的内在时钟，偏了多少？」

通过三轮不同时长的时间感知测试，量化用户的主观时间与客观时间的偏差，生成时间人格报告并匹配历史人物。

## 核心玩法

- 三轮测试，每轮时长不同（5–20 秒区间，相邻差值 ≥4 秒，顺序打乱）
- 用户在动效背景下凭直觉感受时间，觉得「到了」就点击屏幕
- 每轮动效主题不同，但视觉风格统一（浅色系线条/几何/点阵）
- 关键：动效无节奏、无周期，不能被用作计时参照

## 三轮动效主题

| 轮次 | 名称 | 形态 | 特征 |
|------|------|------|------|
| 1 | 涟漪 Ripple | 缓慢漂浮的细线段 | 20 条线段，无规则方向漂移，opacity 0.08–0.2 |
| 2 | 脉冲 Pulse | 同心圆扩散消失 | 5 个圆从随机位置生成并扩散淡出 |
| 3 | 漂移 Drift | 点阵网格随机微偏移 | 12×16 网格点在原位做随机位移 |

所有动效统一用 rgba(26,26,26, 低 opacity) 在浅色背景上绘制。

## 历史人物匹配系统（九宫格）

### 判定逻辑

```
平均偏差率 = mean(各轮 (elapsed - target) / target × 100)
波动度 = max(各轮偏差率) - min(各轮偏差率)

快慢维度（elapsed < target → 感觉时间快 → 偏差率为负）：
  偏差率 < -10%  → 偏快
  -10% ~ +10%   → 居中
  > +10%        → 偏慢

稳定维度：
  波动度 < 12%   → 精密稳定
  12% ~ 28%     → 中等波动
  > 28%         → 混沌波动
```

### 九位人物

| 快慢 \ 稳定性 | 精密稳定 | 中等波动 | 混沌波动 |
|---|---|---|---|
| **偏快** | 拿破仑 | 特斯拉 | 毕加索 |
| **居中** | 康德 | 诸葛亮 | 莫扎特 |
| **偏慢** | 达尔文 | 村上春树 | 李白 |

### 人物肖像

放置于 `portraits/` 文件夹，使用文生图模型生成统一风格黑白肖像。

统一规格：
- 1024×1024 px，纯黑白
- 中性灰背景 #9a9a9a，subtle paper grain
- 上左 30° 软光，gentle fill
- 正面半身像，目光直视
- LIFE 杂志风，深调域，眼部锐利对焦

文件命名：napoleon.jpg / tesla.jpg / picasso.jpg / kant.jpg / zhuge.jpg / mozart.jpg / darwin.jpg / murakami.jpg / libai.jpg

prompt 文件位于 `portraits/*.txt`，详见 `portraits/README.md`。

### 粒子聚合文字动效

- 结果揭晓时，canvas 先渲染人物名字获取像素坐标
- 300 个粒子从随机位置出发，通过 ease-out cubic 缓动归位到文字轮廓
- 90 帧完成聚合，之后在目标点 ±0.8px 微幅抖动

### SVG 钟表头像框

- 结果页人物肖像嵌入 SVG 圆形钟表框中
- 外圈刻度线 + 时标数字，中心为圆形裁剪的人物照片
- 头像加载失败时显示首字母占位符

## 交互细节

1. CSS `overscroll-behavior: none` + `touch-action: manipulation` 禁止误触
2. Web Audio API 预加载提示音（开始 880Hz，结束 660Hz）
3. `performance.now()` 高精度计时
4. 时长生成：5–20 秒随机取三值，相邻差 ≥4 秒，不按升序排列
5. 页面过渡动画：opacity + translateY，0.8s cubic-bezier(0.4, 0, 0.2, 1)
6. 保存图片：html2canvas 截取结果卡片 → canvas.toBlob → 触发下载
7. 姓名 canvas 预渲染：`freezeNameCanvasForCapture()` 防止跨域污染

## 响应式设计

- 移动优先（375px 基准）
- 断点：768px（平板）、1024px（桌面）
- 结果页卡片 max-width 420px，居中约束
- 桌面端通过 max-width + margin auto 保持阅读舒适度
- 全页面 CSS 变量 `--vh` 处理移动端视口高度

## 技术栈

- 纯 HTML + CSS + Vanilla JS（单文件 H5 + 外联 app.js）
- Google Fonts: Noto Serif SC + Playfair Display + Inter
- html2canvas 1.4.1（CDN）
- Canvas 2D API 绘制动效和粒子聚合
- SVG 内联（钟表头像框）
- 移动端优先，PC 端通过 max-width 约束
